# PCのDonkeyUIの起動

## PCでDonkeyUIを起動

```
donkey ui
```

## DonkeyCarの起動を試してみる

![](./img/donkeyui_006.jpg)


DonkeyCarが起動すると、`http://donkeypi.local:8887` にブラウザで接続することで車載カメラなどが表示されます。
